Module Story.StoryPlot

StoryPlot has the type of a string binary tree that holds the main plot.

type plot = string tree
val construct_plot : string list -> plot

construct_plot lst is a plot tree from a given list of strings. Requires: list must be in preorder.

val make_decision : plot -> string -> Character.Character.character -> plot

make_decision pl dec player user will make a decision between two choices, the function will returns the corresponding subtree for the duration of the rest of the game. Requires: dec must be either "A" or "B"

val out_put : plot -> unit

out_put pl prints out the string of the current plot along with the two possible choices offer to the user and/or the notion that the game is now ended with "End"
